🧮 A Matemática das Senhas

Fórmula Fundamental

N = CL

  • N = Número total de combinações possíveis
  • C = Tamanho do alfabeto (caracteres disponíveis)
  • L = Comprimento da senha

Exemplo Prático

Senha com 8 caracteres usando 95 caracteres possíveis:

95⁸ = 6,63 × 10¹⁵ combinações

Tempo para quebrar: ~77 dias (1 bilhão tentativas/segundo)
